# 📖 Overview

Connect Four Ultimate is a modern web-based implementation of the classic Connect Four strategy game.

The project transforms the traditional board game into a premium gaming experience by integrating modern UI/UX principles, intelligent AI opponents, player statistics, achievement systems, leaderboard tracking, responsive design, multiple themes, sound effects, and immersive animations.

Whether you want to challenge an intelligent AI opponent or compete against a friend in local multiplayer mode, Connect Four Ultimate provides an engaging experience across desktop, tablet, and mobile devices.

# 🎮 How To Play

1. Enter your player name.
2. Select an avatar.
3. Choose:

   * Play vs AI
   * Play vs Player
4. Select difficulty level.
5. Start the match.
6. Drop discs into columns.
7. Connect four pieces:

   * Horizontally
   * Vertically
   * Diagonally
8. Win the game and earn achievements.
